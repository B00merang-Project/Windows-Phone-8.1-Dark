# Windows Phone 8.1 Dark

Dark version of the Windows Phone 8.1 theme

### Manual installation

Extract the zip file to the themes directory i.e. `/home/USERNAME/.themes`

### Requirements

- GTK+ 3.20 or above
- Murrine and Pixmap theme engines
